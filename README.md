## üíª Project

A simple CRUD implementation with TypeORM in Node.js. API sever implemented with Express.
<br>

Base URL: `http://localhost:3333`


#### CREATE
`POST:  /user`<br>
`{ "name": "foo" }`
<br>

#### READ
`GET:  /user` - All<br>
`GET:  /user/1` - Specific
<br>

#### UPDATE
`PUT:  /user`<br>
`{ "id": 1 "name": "foo bar" } `
<br>

#### DELETE
`DELETE:  user/`<br>
`{ "id": 1 } `
<br>
